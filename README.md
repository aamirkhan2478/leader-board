# Leader Board

> In this website you will add scores and also see your scores.

## Built With

- HTML
- CSS
- JavaScript
- Webpack

## Getting Started

### Run this command to install dependencies

> npm install

### Run this command to run server

> npm start

### Run this command to build your webpack

> npm run build
